# gproxy

http/https 调试工具,可以用来取代 Charles

## 特性
- [ ] http (pure) proxy
- [ ] https proxy
- [ ] https rootca
- [ ] request/response filter
- [ ] 请求录制和重放
- [ ] 限速,模拟2G/3G/4G
- [ ] load-balancing(可用于反向代理) 
- [ ] websocket frames
- [ ] h2 stream view
- [ ] socks5

## Build from source

```bash
cd cmd

go build -o gproxy
```

## 使用

开发中...

```bash

cmd/gproxy

```
